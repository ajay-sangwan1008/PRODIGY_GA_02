# PRODIGY_GA_02 - Text-to-Image Generation using Stable Diffusion

This project was completed as part of the **Generative AI Internship at Prodigy InfoTech**.

## Project Overview

The objective of this task is to generate high-quality images from natural language prompts using the Stable Diffusion model with the Hugging Face Diffusers library.

## Technologies Used

- Python
- PyTorch
- Hugging Face Diffusers
- Transformers
- Google Colab

## Files

- `Stable_Diffusion_Text_to_Image.ipynb` – Complete implementation of text-to-image generation.
- `requirements.txt` – Required Python packages.
